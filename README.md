# Guttr

A 100% private comic book reader. Your files never leave your browser.

## Features

- Open CBR (RAR) and CBZ (ZIP) files
- Drag & drop or tap to select files
- Page navigation with swipe gestures and keyboard shortcuts
- Fit-to-width and fit-to-height scaling modes
- LTR / RTL reading direction
- Thumbnail grid for quick navigation
- Pinch-to-zoom with smart swipe gating
- Dark mode (follows system preference)
- Installable as a PWA — works offline

## Privacy

No data is sent to any server. No analytics, no tracking, no cookies. All comic processing happens entirely in your browser.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
